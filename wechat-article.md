### 使用方法

#### 方法一：拖拽转换（最简单）
1. 下载工具后，打开文件夹
2. 把 PDF 文件拖到 `pdf2word.exe` 上
3. 松开鼠标，自动开始转换
4. 转换完成后，同目录下会生成一个 `.docx` 文件

#### 方法二：右键菜单
1. 运行 `install-context-menu.bat`（只需一次）
2. 右键点击任意 PDF 文件
3. 选择 "ConvertToWord"
4. 坐等转换完成

#### 方法三：批量转换
如果有一堆 PDF 要转换：
```
pdf2word.exe --batch ./pdf 文件夹 --output ./word 文件夹
```
